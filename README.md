# ⚡ Electricity Theft Detection using Machine Learning

## 📌 Overview

This project uses machine learning to detect electricity theft based on consumption patterns.

Electricity theft causes major losses to power companies. This project helps identify suspicious users by analyzing their electricity usage.

---

## 🎯 Features

* Generates synthetic electricity consumption data
* Detects abnormal usage patterns
* Uses machine learning models:

  * Logistic Regression
  * Random Forest
* Provides accuracy and confusion matrix
* Simple data visualization

---
